# Pygame OOP - Sıra Tabanlı Oyun Projesi

Bu proje, **ULUTEK Maker Çocuk Atölyesi** bünyesinde verilen Python derslerinde, öğrencilere **Nesne Tabanlı Programlama (OOP)** mantığını ve Pygame kütüphanesini temel düzeyde öğretmek amacıyla geliştirilmiş bir eğitim materyalidir.

## 📝 Proje Hakkında

Proje, basit bir sıra tabanlı (turn-based) oyun mekaniğine sahiptir. Kod yapısı, öğrencilerin sınıflar (classes), nesneler (objects) ve oyun döngüsü (game loop) gibi kavramları somut örnekler üzerinden görmesini sağlar.

### Öne Çıkan Özellikler:
* **OOP Temelli Yapı:** Karakterler ve oyun bileşenleri sınıflar kullanılarak tasarlanmıştır.
* **Sıra Tabanlı Mekanik:** Oyuncuların sırayla hamle yapabildiği temel bir mantık kurgulanmıştır.
* **Eğitim Odaklı:** Kod okunabilirliği ve sadelik ön planda tutulmuştur.

## 🛠 Kullanılan Teknolojiler

* **Dil:** Python 3.x
* **Kütüphane:** Pygame

## 🚀 Kurulum ve Çalıştırma

Projeyi yerel makinenizde çalıştırmak için aşağıdaki adımları takip edebilirsiniz:

1.  **Depoyu Klonlayın:**
    ```bash
    git clone [https://github.com/Risenmayfall/Pygame.git](https://github.com/Risenmayfall/Pygame.git)
    ```
2.  **Gerekli Kütüphaneyi Yükleyin:**
    Pygame kütüphanesinin yüklü olduğundan emin olun:
    ```bash
    pip install pygame
    ```
3.  **Oyunu Başlatın:**
    ```bash
    python pygame_oyun.py
    ```
